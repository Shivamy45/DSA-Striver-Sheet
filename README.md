## 🗂️ How I Structure Solutions

Each topic folder is organized to keep practice consistent and easy to review.

```text
topic-folder/
├── main.cpp
├── input.txt
└── output.txt
```

- `main.cpp` contains the C++ implementation for the topic or problem set.
- `input.txt` stores sample input used during local testing.
- `output.txt` stores expected or observed output for quick verification.
- Folder names follow the sheet order, such as `s1-t1-cppBasic/` and `s13-t3-Binary-tree-hard/`.

> Repository snapshot: **46 Striver topic folders** plus LeetCode contest and assignment folder for temporary practice.
